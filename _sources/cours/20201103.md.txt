### Javascript boucle WHILE

boucle WHILE et exercices

je declare une variable var i=1 pour commance la boucle par 1
je met comme parametre tant que i <= 10 continue la boucle
c++ ancremente la variable i par 1  

```javascript
var i = 1;
 
while (i <= 10) {
  console.log(i);
  i++;
}
```
